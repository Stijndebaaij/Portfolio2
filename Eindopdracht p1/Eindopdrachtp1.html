Bij deze opdracht was het de bedoeling om een website te bouwen naar een eigen gekozen onderwerp.
Ik had gekozen voor het onderwerp: Autorijschool Hans de Baaij.
Mijn vader heeft namelijk een eigen rijschool, dus was het mijn idee om een "nieuwe" website voor hem te bouwen.

De verschillende programmeertalen die ik heb gebruikt tijdens het bouwen van deze website zijn:
- HTML
- CSS

Het bouwen van deze website was erg lastig voor mij, omdat ik nog niet heel veel had geleerd.
Ik wilde er een mooie website van maken dus heb er erg veel moeite mee gehad.
Uiteindelijk ben ik best tevreden over het resultaat maar weet ik dat ik inmiddels wel mooiere websites kan maken.
